# Prose Releases

Download the latest [Prose](https://github.com/Tha-vivid-one) for macOS from
[Releases](https://github.com/Tha-vivid-one/prose-releases/releases).

Unlimited on-device dictation for macOS. Private by architecture — audio never
leaves your Mac. Requires macOS 14+ on Apple Silicon.

Updates are delivered in-app via Sparkle (`appcast.xml` in this repo).
